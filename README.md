# Instalation
## requires node.js version 20 or higher
` git clone https://github.com/JewelDesu/tcp-testing-tool.git ` <br>
` cd tcp-testing-tool ` <br>
`npm install ` <br>
### dev mode
` npm run dev `

## compiling for different platforms
### Windows
` nom run dist:win `
### Linux
` npm run dist:linux `
### Mac
` npm run dist:mac `
